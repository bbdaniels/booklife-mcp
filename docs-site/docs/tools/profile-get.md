# profile_get

Get your reading profile with preferences and patterns.

## Parameters

None required.

```json
{}
```

## Response

Returns:
- **Format preferences** — Ebook vs audiobook breakdown
- **Top genres** — Most-read genres with counts
- **Most-read authors** — Favorite authors by volume
- **Reading cadence** — Average books per month/week
- **Streaks** — Current and longest reading streaks
- **Completion rate** — Percentage of started books finished

## Use Cases

- Get personalized recommendations based on your profile
- Understand your reading patterns
- Feed the `what_should_i_read` prompt with profile context
